🚀 BASIC USAGE:

1. Install dependencies:
   install.bat
   OR
   pip install -r requirements.txt

2. Run basic scan:
   python temper-data.py -u https://example.com

3. Run with custom output:
   python temper-data.py -u https://target-site.com -o my-results.json

4. Run with more threads:
   python temper-data.py -u https://wordpress-site.com -t 20

🎯 REAL EXAMPLES:

# WordPress site scan
python temper-data.py -u https://wordpress.org

# Joomla site scan  
python temper-data.py -u https://joomla.org

# Custom site with output
python temper-data.py -u https://my-target.com -o security-report.json
